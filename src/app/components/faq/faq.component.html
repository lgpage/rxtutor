Why aren't the marbles for grouped values the same as RxJS marbles?

RxJS Tutor opts to visualize grouped values in a single frame, whereas RxJS marbles (for testing) opts to advance the virtual time by the number of ASCII characters in the group.

From the RxJS docs :

While it can be unintuitive at first, after all the values have synchronously emitted time will progress a number of frames equal to the number of ASCII characters in the group, including the parentheses. e.g. '(abc)' will emit the values of a, b, and c synchronously in the same frame and then advance virtual time by 5 frames, '(abc)'.length === 5 . This is done because it often helps you vertically align your marble diagrams, but it's a known pain point in real-world testing.
